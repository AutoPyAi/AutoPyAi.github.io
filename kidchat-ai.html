Kidchat AI

Empower your child’s curiosity with KidChat AI—where learning meets safe, private, and interactive AI experiences designed just for young minds.

Key Features

  • Local Execution: Runs directly on the user's computer via localhost, ensuring complete privacy and data control.
  • API-Based AI: Cloud AI processing via Google's Gemini API.
  • Local Data Storage: All conversations and exports remain on user's device. User conversations are not stored on external servers.
  • Internet Required: Real-time API calls for AI responses.
  • Why This is Valuable: Privacy Focused: Conversations don't go through third-party servers. Easy to Deploy: No complex server setup required.
  • Educational Value: Visual Learning: Color-coded chat bubbles and structured responses. Step-by-Step Explanations: Formatted mathematical and conceptual explanations.
  • Customizable Teaching: Adjust AI behavior for different learning styles.Interactive Examples: Live calculation demonstrations with values

Perfect For:

School-aged children (5–12 years) eager to explore AI in a safe, engaging way. KidChat AI helps young learners practice math, explore subjects across the curriculum, and build foundational AI literacy—all through friendly, interactive conversations.

    How It Works

    1. Download and unzip the tool
    2. Place your documents in the docs/ folder
    3. Run rag_basic.py
    4. App will be opened in your localhost browser
    5. Type your question — get instant, accurate answers

*Kidchat setting view — User can set model's behavior for the each session

*Question & Answer View — Shows user's conversation with kidchat

*Saved file 's view — Conversation of each session is saved in a text file
